# 本地化翻译

keywords: 翻译,本地化,多语言,localization,i18n

## 执行步骤

1. 读取源语言文件
2. 保护所有花括号占位符（{0}、{1}、{player_name} 等）
3. 翻译 value，保持 key 完全不变
4. 还原占位符
5. 写入目标语言文件

## 约束

- 花括号内的占位符 **绝对不翻译**：{0} {1} {player_name} {item_name} {price} 等
- 游戏术语符合目标语言习惯：
  - 英文: HP, ATK, DEF, STR, AGI
  - 日文: HP, 攻撃力, 防御力, 力, 素早さ
  - 韩文: HP, 공격력, 방어력
- UI 文本简洁，注意目标语言字符宽度（德文比中文长约 30%，日文比中文短约 20%）
- 语气与游戏风格一致（严肃/轻松/中二）
